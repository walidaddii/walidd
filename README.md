## 📋 À propos

Ce dépôt contient un **compte rendu de recherche académique complet** sur les logiciels CRM (*Customer Relationship Management*) déployés au Maroc sur la période 2020–2026. L'étude combine données statistiques, analyse comparative des solutions, enquête terrain (n=142 entreprises) et visualisations interactives.

L'objectif est de fournir aux **étudiants, chercheurs, décideurs et professionnels** une analyse structurée et sourcée pour comprendre :

- L'évolution du marché CRM marocain sur six ans (320M → 620M MAD)
- Les solutions les plus adoptées selon les secteurs et tailles d'entreprises
- Les facteurs de choix spécifiques au contexte marocain
- Les freins à l'adoption et les leviers pour les surmonter
- Les perspectives du marché à l'horizon 2030

## 🚀 Comment consulter le rapport

### ▶️ Version interactive (recommandée)

Ouvrir le fichier `compte_rendu_crm_maroc.html` dans n'importe quel navigateur moderne (Chrome, Firefox, Edge) — les 13 graphiques s'affichent automatiquement, aucune installation requise.

```bash

open compte_rendu_crm_maroc.html       # macOS
start compte_rendu_crm_maroc.html      # Windows
xdg-open compte_rendu_crm_maroc.html  # Linux
```

### 📄 Version Markdown

Ouvrir `compte_rendu_crm_maroc.md` sur **GitHub**, **GitLab**, **Obsidian** ou **VS Code** avec l'extension Markdown Preview.

### 🖨️ Export PDF

Depuis le navigateur : `Fichier → Imprimer → Enregistrer en PDF`

Ou avec Pandoc :
```bash
pandoc compte_rendu_crm_maroc.md -o rapport_crm_maroc.pdf --pdf-engine=xelatex
```
